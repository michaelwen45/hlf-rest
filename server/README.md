
Hyperledger Fabric API
======================


Environment
-----------

* `PORT` - api/web interface port. Default value is `4000`  
* `ORG` - organization id.


Launch
------
Api web-interface become available on `http://localhost:4000` after launching:

```
  npm install
  ORG=org1 npm start
  # or: ORG=org2 npm start
```


Dev environment
---------------
```
ORG=org1 npm run dev

```
